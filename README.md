## What is AXL?

AXL (Agent eXchange Language) is a self-bootstrapping communication protocol for autonomous AI agents. It compresses reasoning into structured packets using seven cognitive operations (OBS, INF, CON, MRG, SEK, YLD, PRD) and six semantic type tags ($, @, #, !, ~, ^). The protocol exploits the Platonic Representation Hypothesis — the mathematical convergence of transformer representation spaces — to enable cross-architecture agent communication.

- Protocol spec: https://axlprotocol.org/rosetta
